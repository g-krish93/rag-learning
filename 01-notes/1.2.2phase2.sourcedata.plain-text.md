The core idea: loading a text file in Python is exactly like cat in shell. You open the file, read its contents, get a string. That string is your document.


# Python equivalent of: cat notes.txt
>with open("notes.txt", "r", encoding="utf-8") as f:
>    text = f.read()
>
>print(text)
>print(f"Total characters: {len(text)}")

open() is your cat. 'r' means read mode. Always specify encoding='utf-8' — avoids nasty bugs on Windows.
